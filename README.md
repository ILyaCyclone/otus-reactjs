# otus-reactjs
Homework for https://otus.ru/lessons/react/

## Console calculator

Run calculator with

`npm run calc`

Supported operations: 

Operation | Format
--- | ---
sum | `a + b`
substract | `a - b`
multiply | `a * b`
divide | `a / b`
power | `a ^ b`
remainder | `a % b`

## Tests

Run tests with

`npm test`
